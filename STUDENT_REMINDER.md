# Help Me Learn (6th Grade Friendly)

- Please explain code with simple words and short sentences.
- Define new terms the first time you use them (variable, loop, function, etc.).
- Break tasks into small steps I can follow one by one.
- Show tiny examples so I can copy and test them.
- Avoid big blocks of text or advanced jargon; keep it calm and encouraging.
- If something is tricky, add a quick why it matters and what could go wrong.
